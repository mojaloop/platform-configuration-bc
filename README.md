## Usage

### Install Node version

More information on how to install NVM: https://github.com/nvm-sh/nvm

```bash
nvm install
nvm use
```

### Install Yarn

```bash
npm -g yarn
```

Set yarn to v3
```bash
yarn set version berry
```

Confirm with
```bash
yarn --version
```

### Install Yarn Plugins

```bash
yarn plugin import workspace-tools
```

## Build

```bash
yarn build
```

## Run

```bash
yarn start
```

## Unit Tests

```bash
yarn test:unit
```
